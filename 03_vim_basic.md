# 🕐 3. 리눅스 텍스트 편집기 vi/vim 기본

## 🎯 목표

리눅스 텍스트 편집기 vi/vim을 사용해 **파일을 열고, 수정하고, 저장하고, 종료**하는 기본 흐름을 익힌다.

* * *

## 🧩 개요 — vi/vim 이란?

- vi: 전통적인 유닉스 텍스트 에디터, 대부분의 리눅스에 기본 설치
- vim: vi 개선판(Vi IMproved), 문법 강조·다중 언두 등 추가

* * *

## 📁 사전 준비 — 3가지 모드 이해 (필수 개념)

| 모드 | 진입 방법 | 특징 |
|------|-----------|------|
| NORMAL | 시작 시 / `Esc` | 이동·복사·삭제 명령어 사용 가능 |
| INSERT | `i`, `a`, `o` | 실제 텍스트 입력 |
| COMMAND | `:` | 저장·종료·검색 등 명령 실행 |

### 필수 단축키 (NORMAL / COMMAND 모드)

**NORMAL 모드 — 이동**

| 키 | 동작 |
|----|------|
| `h` `j` `k` `l` | 좌·하·상·우 이동 |
| `w` / `b` | 다음 / 이전 단어 이동 |
| `0` / `$` | 줄 맨 앞 / 맨 뒤 |
| `gg` / `G` | 파일 맨 처음 / 맨 끝 |

**NORMAL 모드 — 편집**

| 키 | 동작 |
|----|------|
| `i` | 커서 앞 INSERT |
| `a` | 커서 뒤 INSERT |
| `o` | 아래 줄 새 INSERT |
| `dd` | 현재 줄 삭제 |
| `yy` | 현재 줄 복사 |
| `p` | 붙여넣기 |
| `u` | 실행 취소 (Undo) |
| `Ctrl+r` | 재실행 (Redo) |

**COMMAND 모드 — 저장·종료**

| 명령 | 동작 |
|------|------|
| `:w` | 저장 |
| `:q` | 종료 (미저장 파일 있으면 오류) |
| `:wq` | 저장 후 종료 |
| `:q!` | 강제 종료 (변경 무시) |
| `ZZ` | `:wq` 와 동일 (NORMAL에서) |

**COMMAND 모드 — 검색**

| 명령 | 동작 |
|------|------|
| `/단어` | 아래 방향 검색 |
| `n` / `N` | 다음 / 이전 검색 결과 |

* * *

## 🧩 실습 1 — 파일 열기 · 수정 · 저장

새 파일을 열고 내용을 작성해봅니다.

```bash
# vim 설치 (최초 1회)
apt update && apt install -y vim

# 새 파일 열기
vim note.txt

# 1. i 를 눌러 INSERT 모드 진입
# 2. 아래 내용 입력
Hello, vim!
This is my first file.

# 3. Esc 눌러 NORMAL 모드로 복귀
# 4. :wq 로 저장 후 종료
```
확인:
```bash
cat note.txt
```

* * *

## 🧩 실습 2 — 기존 파일 수정

이전 실습에서 만든 파일을 열어 수정해봅니다.

```bash
# 02.md 실습에서 만든 test.txt 열기 (또는 note.txt)
vim test.txt

# 1. G 로 파일 끝으로 이동
# 2. o 로 새 줄 INSERT
# 3. "edited by vim" 입력
# 4. Esc → :wq
```
확인:
```bash
cat test.txt
```

* * *

## 🧩 실습 3 — 검색

```bash
vim note.txt

# NORMAL 모드에서
/vim    # "vim" 검색
n       # 다음 검색 결과
:q!     # 종료 (수정사항 무시)
```

* * *

## 📘 실습 요약

1. `vim 파일명` 으로 파일 열기
2. `i`, `a`, `o` 로 입력 모드(INSERT) 진입
3. 내용 작성 후 `Esc` 로 일반 모드(NORMAL) 복귀
4. `:wq` 로 저장 후 종료, `:q!` 로 저장 없이 강제 종료
5. `dd` (줄 삭제), `yy` (줄 복사), `p` (붙여넣기)
6. `/단어` (검색), `n` (다음 결과)
