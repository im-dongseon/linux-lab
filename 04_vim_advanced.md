# 🕐 4. 리눅스 텍스트 편집기 vi/vim 심화

## 🎯 목표

vim을 실무에서 효율적으로 사용하기 위해 **.vimrc 설정**, **visual 모드**, **쉘 스크립트 작성에 필요한 단축키**를 익힌다.

* * *

## 🧩 개요 — .vimrc 개요 및 필요성

- vim은 기본 설정이 최소화되어 있어, 실사용에 불편함이 있습니다.
- `~/.vimrc` 파일을 만들면 vim 시작 시 자동으로 설정이 적용됩니다.
- 한 번 설정해두면 어떤 파일을 열어도 동일한 환경을 유지할 수 있습니다.

* * *

## 🧩 실습 1 — 대중적인 .vimrc 설정 + 설명

`~/.vimrc` 파일을 열어 다음 설정을 적용하고 효과를 확인합니다.
```bash
vim ~/.vimrc
```

**대중적인 .vimrc 설정 목록**

| 설정 | 코드 | 설명 |
|------|------|------|
| 줄 번호 표시 | `set number` | 각 줄 앞에 번호 표시 |
| 상대 줄 번호 | `set relativenumber` | 현재 커서 기준 상대 번호 (이동 편리) |
| 개행·공백 표시 | `set list` | 탭·공백·줄 끝 문자 시각화 |
| 개행 문자 지정 | `set listchars=eol:↵,tab:»·,trail:·` | 줄 끝(↵), 탭(»·), 후행 공백(·) 표시 |
| 인코딩 설정 | `set encoding=utf-8` | 한글 등 유니코드 깨짐 방지 |
| 탭 폭 설정 | `set tabstop=4` | 탭 문자를 4칸으로 표시 |
| 탭→스페이스 | `set expandtab` | Tab 키 입력 시 스페이스로 대체 |
| 자동 들여쓰기 | `set autoindent` | 엔터 후 이전 줄 들여쓰기 유지 |
| 스마트 들여쓰기 | `set smartindent` | 코드 구조에 맞게 자동 들여쓰기 |
| 검색 하이라이트 | `set hlsearch` | 검색 결과 강조 표시 |
| 실시간 검색 | `set incsearch` | 입력하는 동안 실시간 검색 |
| 구문 강조 | `syntax on` | 파일 형식에 따라 색상 강조 |

아래의 설정을 `.vimrc` 파일에 입력하고 `:wq`로 저장합니다.
```vim
set number
set tabstop=4
set expandtab
set autoindent
set hlsearch
syntax on
set encoding=utf-8
set list
set listchars=eol:↵,tab:»·,trail:·
```

> **💡 중요 (.vimrc 적용 위치)**
> `vim`은 시작할 때 사용자 설정 파일을 기본적으로 `~/.vimrc` 에서 찾습니다.
> `~/workspace` 경로(공유 볼륨)에서 `.vimrc` 파일을 생성하였다면 vim 실행 시 설정이 제대로 먹히지 않습니다. `.vimrc` 는 항상 사용자 홈 디렉토리(`~` 또는 `/root`)에 위치해야 합니다.
> 만약 현재 경로에서 만들었다면 아래 명령어로 홈 디렉토리에 복사하거나 이동해주세요.
> `cp .vimrc ~/` 또는 `mv .vimrc ~/.vimrc`

## 🧩 실습 2 — visual 모드 사용하기

vim의 visual 모드는 마우스 드래그처럼 블록을 지정하여 편집할 때 유용합니다.

| 모드 | 진입 | 용도 |
|------|------|------|
| visual (문자) | `v` | 문자 단위 블록 선택 |
| visual line | `V` | 줄 단위 블록 선택 |
| visual block | `Ctrl+v` | 직사각형 블록 선택 (멀티라인 편집) |

**멀티 라인 복사-붙여넣기 — 방법 1: visual line 모드 사용**
```
1. V 로 줄 단위 선택 시작
2. j 로 여러 줄 선택
3. y 로 복사
4. 원하는 위치로 이동 후 p 로 붙여넣기
```

**멀티 라인 복사-붙여넣기 — 방법 2: Nyy + p 단축키 사용 (visual 모드 없이)**
```
# NORMAL 모드에서
3yy   # 현재 줄부터 3줄 복사 (N줄 복사: Nyy)
p     # 커서 아래에 붙여넣기
P     # 커서 위에 붙여넣기
```
> 방법 1은 범위를 눈으로 확인하며 선택할 때, 방법 2는 줄 수를 정확히 알 때 빠르게 사용합니다.

**visual block으로 멀티 라인 앞에 # 추가 (주석 처리)**
```
1. Ctrl+v 로 블록 모드 진입
2. j 로 여러 줄 선택
3. I (대문자 i) 로 줄 앞 INSERT 모드
4. # 입력 후 Esc
-> 선택한 모든 줄에 # 이 추가됩니다.
```

## 🧩 실습 3 — 라인 이동 단축키 심화

| 키 | 동작 |
|----|------|
| `:{n}` | n번째 줄로 이동 (예: `:10`) |
| `Ctrl+d` | 반 페이지 아래로 |
| `Ctrl+u` | 반 페이지 위로 |
| `Ctrl+f` | 전체 페이지 아래 |
| `Ctrl+b` | 전체 페이지 위 |
| `%` | 괄호 짝으로 이동 (`(`, `[`, `{` 에서) |

* * *

## 🧩 실습 4 — 쉘 스크립트 작성에 유용한 설정·단축키

| 키 / 명령 | 동작 | 쉘스크립트 활용 |
|-----------|------|----------------|
| `:set filetype=sh` | 파일 형식을 sh로 지정 | bash 문법 강조 적용 |
| `>>` / `<<` | 들여쓰기 1탭 추가/제거 | if·for 블록 정리 |
| `=G` | 현재 위치부터 파일 끝까지 자동 들여쓰기 | 스크립트 전체 정렬 |
| `:!bash %` | 현재 파일을 bash로 바로 실행 | 저장 후 나가기 없이 테스트 |
| `Ctrl+n` | 자동완성 (INSERT 모드) | 변수명·함수명 반복 입력 시 |
| `:syntax on` | 구문 강조 수동 활성화 | 색상으로 오류 위치 파악 |

* * *

## 📘 실습 요약

1. `~/.vimrc` 를 통해 vim 환경을 사용자 맞춤 설정 가능
2. `V`, `Ctrl+v` 를 활용한 멀티 라인 블록 지정, 복사/붙여넣기 및 주석 처리
3. `:{n}` 또는 `Nyy` 등 숫자와 조합하여 여러 줄을 한 번에 제어
4. `>>`, `=G`, `:!bash %` 등을 활용하여 코드 포매팅 및 즉각 실행 가능
